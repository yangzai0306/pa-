# 图片文字排版合成工具 使用说明
## 工具介绍
这款图片文字排版合成工具是一款轻量化的网页端应用，无需安装本地软件，可快速实现图片上传、文字添加与排版，并将排版好的内容合成为一张高清图片下载。工具支持拖拽排序、文字样式自定义（字体大小、对齐方式），且能自动控制合成图片大小在500KB~3MB之间，满足大多数场景的使用需求（如社交分享、图文编辑、素材制作等）。

工具核心特性：
1. 纯网页端操作，无需安装，打开即可使用；
2. 支持多图片上传、文字自由添加，内容可拖拽调整顺序；
3. 文字支持字体大小（12px~36px）、对齐方式（左/中/右）自定义；
4. 合成图片自动压缩，大小控制在500KB~3MB，兼顾画质与体积；
5. 操作简单，预览直观，生成后可直接下载。

## 使用步骤
### 一、基础操作
#### 1. 上传图片
点击页面上方的「上传图片」按钮，在弹出的文件选择窗口中，选择本地需要添加的图片（支持多选），确认后图片会自动添加到「编辑预览区」。
- 上传的图片会自动适配预览区宽度（最大800px），保证排版整洁；
- 可多次点击「上传图片」按钮，分批添加多张图片。

#### 2. 添加文字
点击页面上方的「添加文字」按钮，编辑预览区会新增一个文字编辑卡片，支持以下自定义操作：
- **输入文字**：点击文字区域（默认显示「请输入文字内容...」），直接输入/修改需要展示的文字；
- **调整字体大小**：通过文字卡片上方的下拉框选择字号（12px~36px可选）；
- **调整对齐方式**：点击「左对齐」「居中对齐」「右对齐」按钮，设置文字的对齐样式。

#### 3. 拖拽排序
编辑预览区内的图片/文字卡片支持拖拽调整顺序：
1. 将鼠标移至需要调整的卡片上（光标变为「移动」样式）；
2. 按住鼠标左键，拖拽卡片到目标位置后松开，即可完成排序。

#### 4. 删除内容
每张图片/文字卡片的右上角都有红色「×」删除按钮，点击即可删除该卡片内容。

#### 5. 清空全部
点击页面上方的「清空全部」按钮，确认后会清空编辑预览区的所有内容，同时隐藏合成结果区。

### 二、生成并下载合成图片
1. 完成图片、文字的排版后，点击页面上方的「生成并下载」按钮；
2. 工具会自动处理内容（隐藏编辑工具栏、删除按钮等无关元素），生成合成图片；
3. 生成完成后，页面下方的「合成结果」区域会显示最终图片，同时浏览器会自动触发下载（文件名格式：合成图片_时间戳.jpg）；
4. 生成的图片大小会自动控制在500KB~3MB之间，无需手动压缩。

## 注意事项
1. 图片上传仅支持常见图片格式（如JPG、PNG、GIF等），且需保证文件未损坏；
2. 文字编辑区域支持换行（按Enter键）、空格等基础排版，内容会自动适配宽度；
3. 合成图片时，若预览区无内容会提示「请先添加内容」，需至少添加一张图片或一段文字；
4. 建议在Chrome、Edge、Firefox等现代浏览器中使用，避免兼容性问题；
5. 生成的图片为JPG格式，背景默认为白色，若需透明背景需自行处理原始素材。

## 常见问题
Q：上传图片后预览区不显示？
A：请检查图片格式是否为支持的类型，或图片文件是否损坏；若浏览器禁用了JS，需开启后重试。

Q：合成的图片体积超出/低于预期？
A：工具已自动适配体积范围（500KB~3MB），若需更小体积，可适当减少图片/文字内容，或降低图片原始分辨率。

Q：拖拽排序无效？
A：确保鼠标按住卡片主体区域（非按钮/文字编辑区），且预览区有至少两张卡片时排序功能才生效。
